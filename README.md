# Projet DevOps

**Nom:** Nabil
**Prénom:** Fahssi

## Description

Ce projet est un mini projet DevOps qui démontre l'utilisation de Git, GitHub, GitHub Actions, Jenkins, Docker et d'autres outils DevOps.

## Application

Application Java simple qui affiche le message: "Bonjour et bon courage dans votre projet en DevOps"

## Technologies utilisées

- Java
- Maven
- Git / GitHub
- GitHub Actions
- Jenkins
- Docker (optionnel)
